# riichi-mahjong-trainer
Riichi Mahjong discard trainer (front-end only): shanten + ukeire feedback  
纯前端日麻摸切练习器：向听 + 有效进张（ukeire）反馈

## English Description
A simple Riichi Mahjong discard training tool:
- Single-player
- No calls / no scoring / no yaku
- Compare: normal-hand shanten first, then ukeire (effective tiles count)

### Try it online
Open the GitHub Pages link:  
https://github.com/DannyNi-Eng/riichi-mahjong-trainer/deployments/github-pages

### Controls
- Hover on a tile to lift it
- Left click to discard

### Local run (optional)
Use any static server (recommended):
- VSCode Live Server
- `python -m http.server 8000`

## 中文
一个简单的日麻摸切练习网页工具：
- 单人练习
- 不包含吃/碰/杠、副露、役种、点数等对局要素
- 对比规则：先比较普通手向听数，再比较有效进张枚数（ukeire）

### 在线试玩
打开 GitHub Pages 链接：  
https://github.com/DannyNi-Eng/riichi-mahjong-trainer/deployments/github-pages

### 操作方式
- 鼠标悬停在牌上：牌会抬起
- 鼠标左键点击：弃牌

### 本地运行（可选）
用任意静态服务器打开即可（推荐）：
- VSCode Live Server
- `python -m http.server 8000`
